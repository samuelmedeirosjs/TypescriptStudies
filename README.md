# TypeScript Studies

> Status: Em andamento 🚧

Repositório dedicado ao meu aprendizado em **TypeScript**, contendo exemplos práticos, anotações e pequenos experimentos organizados por assunto.

O objetivo é consolidar conceitos fundamentais da linguagem e construir uma base sólida para aplicações modernas, especialmente no ecossistema Front-end.

---

## 📚 Conteúdo

Os estudos estão organizados por pastas, seguindo uma progressão lógica de aprendizado que atualmente se encontra assim:

- `01-types-basics` – Tipos primitivos, inferência e tipagem explícita
- `02-interfaces-types` – Interfaces, types e suas diferenças(em breve)

Cada pasta pode conter:
- Arquivos `.ts` com exemplos práticos
- Arquivos `.js` com código compilado
